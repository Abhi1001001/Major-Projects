🧑‍💻 Abhishek Vishwakarma – Portfolio Website

Welcome to my personal web developer portfolio, built using modern React tooling and libraries. This website is a fully responsive, animated showcase of my skills, projects, experience, education, and contact information.

Abhishek-portfolio

🚀 Tech Stack

  • React (with Vite)
  • React Router DOM – Smooth page navigation
  • GSAP (GreenSock) – Elegant and performant animations
  • SVGR – Optimized SVG integration
  • Custom React Components – Modular structure
  • Dark/Light Mode – Seamless UI toggle
  • CSS / Flexbox / Grid– Clean and responsive layout

✨ Features

  • ✅ Single Page Application (SPA) with client-side routing
  • ✅ Dark and Light Theme Toggle for better accessibility
  • ✅ GSAP Animations for smooth transitions and element reveals
  • ✅ Optimized SVGsusing SVGR for performance and clarity
  • ✅ Fully Responsive Design – Mobile-first and adaptable to all screen sizes
  • ✅ Modular Codebase with reusable components and clean structure

📁 Pages Included

Page Description
Home Landing section with a personal intro and call to action

🛠️ Getting Started

1. Clone the Repository

git clone https://github.com/Abhi1001001/Major-Projects/abhishek-portfolio-3.0.git cd abhishek-portfolio-3.0

2. Install Dependencies

npm install

3. Run the Development Server

npm run dev

4. Build for Production

npm run build
